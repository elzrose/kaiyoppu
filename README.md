# Kaiyoppu

Kaiyoppu is a secure, role-based digital hiring platform that enables trusted interactions between workers and hirers through QR-based identity verification.

## 🚀 Features
- Google Authentication (Firebase)
- Role-based access (Worker / Hirer / Admin)
- QR-based worker identity generation
- Worker verification system for hirers
- Admin approval system to ensure trust
- Real-time data handling with Firestore
- Clean and user-friendly interface

## 🛠️ Tech Stack
- React (Vite)
- Firebase (Authentication + Firestore)
- JavaScript
- HTML5 / CSS3
- Git & GitHub
- Vercel (Deployment)
